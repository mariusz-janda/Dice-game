
# Dice Game 🎲

Gra w JavaScript, w której gracz rywalizuje z komputerem.

## Zasady:
- Rzucasz kostką
- Jeśli wypadnie 1 – tracisz punkty
- Wygrywa pierwszy do 20 punktów

## Technologie:
- HTML
- CSS
- JavaScript

## Funkcje:
- System punktacji
- Prosty przeciwnik (AI)
- Historia rzutów
- Reset gry
